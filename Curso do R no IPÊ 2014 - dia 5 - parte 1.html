Incluindo informao de outra varivel

Imagine que cada ponto no mapa possua a informao de uma varivel quantitativa qualquer. Ns podemos inclu-la no mapa usando pontos cujo tamanho proporcional ao valor da varivel de interesse! Para isso vamos criar uma varivel qualquer, ao acaso, apenas para ver como fica, ok?

abund<-rpois(nrow(dados), 5) #Cria a varivel que queremos


plot(wrld_simpl, xlim=c(-90, -30), ylim=c(-60, 20),
axes=T, col="gray")

points(dados, pch=16, col="red", cex=abund/5)

Ns usamos o argumento cex para determinar os tamanhos dos pontos; dividimos por 5, porque os nmeros estavam gerando bolinhas muito grandes.

Mas tem uma coisa chata: fica difcil, com as bolinhas sobrepostas, saber onde est o centro dela. Ento vamos adicionar pontos pequenos:

plot(wrld_simpl, xlim=c(-90, -30), ylim=c(-60, 20),
axes=T, col="gray")

points(dados, pch=16, col="red", cex=abund/5)

points(dados, pch=16, cex=0.1)

Agora sim! :)

Extraindo dados de layers

Uma das coisas mais bacanas ao usar dados de informao geogrfica no R que temos uma tremenda capacidade de lidar com os dados por trs dos mapas, como em layers com variveis ambientais, por exemplo. Vamos, ento, nos divertir um pouco com isso.

Para facilitar, vamos usar os dados que vm junto do pacote dismo, mas a mesma funo pode ser usada para acessar dados no seu computador, ok? Vamos l:

files<-list.files(path=paste(system.file(package="dismo"),
"/ex", sep=""), pattern="grd", full.names=T)

Vamos aos detalhes:

  • a funo list.files faz uma lista dos arquivos de uma pasta que atendem um certo critrio;
  • o argumento path determinar a pasta escolhida (com seus dados, coloque o endereo da pasta no mesmo formato usado na setwd);
  • o argumento pattern determina o tipo de arquivo que vamos ler (neste caso, gridfiles);
  • o argumento full.names serve para vermos o endereo completo da pasta quando chamamos o objeto;

Extraindo os valores:

Vamos, agora, extrair os valores das duas variveis ambientais nos nossos layers a partir dos nossos registros de coleta:

Criando pontos ao acaso e comparando o clima com os registros

Vamos criar 500 pontos ao acaso na nossa regio de estudo, olharmos para eles rapidamente no mapa e depois extrairmos as mesmas variveis ambientais para eles tambm:

pontos.acaso<-randomPoints(clima, 500)

plot(clima$bio1)
points(pontos.acaso)

clima.acaso<-extract(clima, pontos.acaso)

E vamos agora comparar tudo com um boxplot:

boxplot(clima.pontos[,1]/10, clima.acaso[,1]/10, range=0)
