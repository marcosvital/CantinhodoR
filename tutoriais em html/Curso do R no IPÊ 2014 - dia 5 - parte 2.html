Explorando a correlao entre as variveis

Uma coisa fundamental em uma regresso mltipla utilizar variveis explicativas que no estejam correlaciomnadas entre si. Podemos avaliar se isto acontece com nossos dados calculando um coeficiente de correlao entre as variveis de nosso interesse.

Mas como sou chato, quero algo que me permita olhar para as correlaes ao mesmo tempo que vejo em grficos como os dados se relacionam.

O nosso ponto de partida a funo pairs:

pairs(dados[2:6])

Que j bem legal, mas pode ser melhor aproveitada. Que tal colocar coisas teis na diagonal principal e no painel superior?

Ns vamos copiar e colar um trecho do exemplo do help da funo, fazer uma modificao bem pequena, e executar. Se no quiser mexer com isso diretamente, basta executar o cdigo abaixo:

panel.hist <- function(x, ...)
{
    usr <- par("usr"); on.exit(par(usr))
    par(usr = c(usr[1:2], 0, 1.5) )
    h <- hist(x, plot = FALSE)
    breaks <- h$breaks; nB <- length(breaks)
    y <- h$counts; y <- y/max(y)
    rect(breaks[-nB], 0, breaks[-1], y, col = "cyan", ...)
}


panel.cor <- function(x, y, digits = 2, prefix = "", cex.cor, ...)
{
    usr <- par("usr"); on.exit(par(usr))
    par(usr = c(0, 1, 0, 1))
    r <- abs(cor(x, y))
    txt <- format(c(r, 0.123456789), digits = digits)[1]
    txt <- paste0(prefix, txt)
    if(missing(cex.cor)) cex.cor <- 0.8/strwidth(txt)
    text(0.5, 0.5, txt, cex = 1)
}

Beleza, agora podemos re-escrever nossa funo, e criar um grfico bem til para se explorar os padres de correlao entre as variveis (com um histograma de cada uma de brinde no meio!).

Vamos l:

pairs(dados[2:6], 
diag.panel=panel.hist, upper.panel=panel.cor)

Vejamos:

  • o argumento diag.panel define o que vai aparecer na diagonal principal da matriz de grficos;
  • o argumento upper.panel define o que aparece no painel superior;
  • panel.hist e panel.cor foram definidos naquele bloco de cdigo logo acima.

Fazendo a regresso mltipla

facinho! Se voc j fez uma regresso mltipla, ver que no tem nada demais aqui:

modelo<-lm(Dens_Praga~Altitude+Pluviosidade+Mata)

Vamos dar uma olhada nos pressupostos?

par(mfrow=c(1,2))

#Homocedasticidade:
plot(modelo$fitted, modelo$residuals,
abline(h=0,lty=2))

#Normalidade:
qqnorm(modelo$residuals)
qqline(modelo$residuals, lty=2)

No est fantstico, mas considero como aceitvel.
