A tal da PCA

Existem diversas funes diferentes no R para calcular uma PCA. Aqui ns vamos usar a funo rda do pacote vegan.

No h muito mistrio: basta executar a funo rda com o argumento scale=T, que faz com que os dados sejam padronizados (uma caracterstica da PCA). Depois, s chamar o summary. Note que o resultado ser um pouco longo, ok?

A est:

dados.amb<-dados[,3:6]

library(vegan)
## Loading required package: permute
## Loading required package: lattice
## This is vegan 2.0-10
resultado.pca<-rda(dados.amb, scale=T)

summary(resultado.pca)

Plotando de maneira rpida

No tem mistrio aqui, vamos usar a funo biplot:

biplot(resultado.pca)

Se quiser plotar outros componentes, voc pode usar o argumento choices. Experimente, por exemplo:

biplot(resultado.pca, choices=c(1,3))  #Plotando os componentes 1 e 3

Incluindo uma varivel categrica no grpafico:

Podemos delimitar os pontos de mata primria e de mata secundria no nosso grfico assim:

biplot(resultado.pca, display="sites")

ordihull(resultado.pca, group=Ambiente,
show="Mata_Primria", col="green")

ordihull(resultado.pca, group=Ambiente,
show="Mata_Secundria", col="red")

Os detalhes:

  • usamos o argumento display="sites" para mostrar apenas os locais de coleta, sem as variveis;
  • usamos a funo ordihull para criar polgonos;

Se quiser, confira tambm o mesmo procedimento acima com a funo ordiellipse.

Brincando com os resultados da PCA

Extraindo os valores dos componentes principais

Esta uma etapa essencial, se quisermos usar os resultados da PCA em outras anlises. bem simples:

pc1<-resultado.pca$CA$u[,1]
pc2<-resultado.pca$CA$u[,2]

Exemplinho rpido com teste T:

t.test(pc1~Ambiente, var.equal=T)
